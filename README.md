# Sample-Language-Compiler
编译原理课程设计，包含词法分析、LL1预测分析、LR1语法分析、DGA优化、算符优先语法分析、采用LR1生成中间代码、中间代码转换为汇编代码。


## 运行环境：
    - python == 3.6
    - pyqt5 == 4.19
    - graphviz
## graphviz 需要安装外部依赖软件，官方下载: https://graphviz.gitlab.io/_pages/Download/Download_windows.html
## 执行 runfile.py 进入界面。
